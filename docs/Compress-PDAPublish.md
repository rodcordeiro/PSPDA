# Compress-PDAPublish

## SYNOPSIS

Gera um arquivo zipado com a build da aplicação, contendo os arquivos informados no campo _files_

## SYNTAX

```
Compress-PDAPublish [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Gera um arquivo zipado com a build da aplicação, contendo os arquivos informados no campo _files_

## EXAMPLES

### Example 1

```powershell
PS C:\> Compress-PDAPublish
```

Gera o arquivo
